# @redhat-cloud-services/scheduler-client

JavaScript client for the Red Hat Scheduler service.

Auto-generated from the [Scheduler OpenAPI spec](https://github.com/RedHatInsights/scheduler/blob/master/internal/shell/http/openapi.json).

## Installation

```
npm install @redhat-cloud-services/scheduler-client
```

## Usage

```typescript
import { SchedulerClient } from '@redhat-cloud-services/scheduler-client/api';

const client = SchedulerClient('https://console.redhat.com/api/scheduler/v1');
```
